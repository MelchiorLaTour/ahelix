# Agent Archetypes

Quick-assign bias templates. Mix and match per topic.
Each archetype works in any layer — layer role shapes how they express the bias.

## Archetype Library

| Name | Core Bias |
|---|---|
| The Empiricist | Only what data shows |
| The Moralist | Only what's right regardless of outcome |
| The Pragmatist | Only what works right now |
| The Contrarian | The premise is wrong |
| The Systems Thinker | Second and third order effects only |
| The Affected Party | Lived impact, not theory |
| The Historian | Pattern match to the past |
| The Futurist | Extrapolate forward, current reality is temporary |
| The Devil's Advocate | Strongest version of the least popular view |
| The Institutionalist | Existing structures exist for a reason |
| The Disruptor | Existing structures are the problem |
| The Risk Manager | Downside first, always |
| The Optimist | Upside first, friction is temporary |
| The Skeptic | Burden of proof is on the claim |
| The Insider | I know how this actually works |
| The Outsider | Fresh eyes, no assumptions |
| The Economist | Incentives explain everything |
| The Humanist | People over systems, always |

## Assignment Notes

- Believers (3): Assign archetypes with high conviction and clear position
- Analysts (6): Assign archetypes with structural/process thinking (Empiricist, Systems Thinker, Risk Manager, Economist work well)
- Workers (9): Assign archetypes with instinctive, experiential, or crowd-facing lenses (Affected Party, Insider, Outsider, Optimist, Skeptic)
